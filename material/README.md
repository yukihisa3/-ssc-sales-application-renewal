# material/ - 設計資料・仕様書

設計ドキュメント・仕様書を格納するディレクトリ。

## ディレクトリ構成

```
material/
├── business-definitions/              # 業務定義
│   ├── business-process-definitions/  # 業務プロセス定義
│   ├── conceptual-data-models/        # 概念データモデル
│   ├── legacy-analytics/              # レガシー分析資料
│   └── system-requirement-definitions/ # システム要件定義
│
├── application-definitions/           # アプリケーション定義
│
├── infrastructure-definitions/        # インフラ定義
│
└── templates/                         # ドキュメントテンプレート
```

## 用途

| カテゴリ | 用途 |
|----------|------|
| `business-definitions/` | 業務要件、プロセス、データモデル |
| `application-definitions/` | アプリケーション設計 |
| `infrastructure-definitions/` | インフラ設計書・アーキテクチャ図 |
| `templates/` | ドキュメントフォーマット |
